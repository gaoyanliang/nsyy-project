## 高压氧系统

### 1. 登记

登记氛围两种：

1. 住院登记
2. 门诊登记

住院登记，直接输入患者住院号，系统自动查询患者信息，完成登记。
门诊登记，人工输入患者信息，完成登记。

登记时，需要确定，开始日期，每日开始时间，每次执行时间 （开始时间不能小于当天）

### 2. 执行

首次执行时需要确认医生已开医嘱，然后签署知情同意书和进行心理指导。 签署完成会创建第一天的执行记录。

之后，每完成一天的执行记录，会创建下一天的执行记录。如果整个执行周期完成，则执行完成。

每天的执行记录，可以选择取消（取消当天，执行周期继续），或取消今后所有（整个执行周期结束）

已经取消的执行周期，如果还未到执行周期的结束时间，可以选择恢复执行


### 3. 扣款

门诊患者 系统不支持扣款。 现场刷就诊卡

扣款时，正式环境 需要的是 开单科室编码/执行科室编码（不是 id）， 测试环境需要的是 id
